# Panduan Ruby on Rails Indonesia

## Referensi
- https://github.com/rails/rails/tree/master/guides

## Kontribusi

```bash
git clone git@github.com:rails-id/guides.git
```

Lakukan perubahan pada panduan di direktori `source/`

```bash
bundle install
rake guides:generate:html
```
